# MCP Dashboard

MCP(Model Context Protocol) 생태계 현황을 보여주는 대시보드입니다.
SIFT Lab(경희대)에서 진행 중인 MCP 공급망 보안(rug-pull 탐지) 연구 과정에서
수집한 메타데이터의 공개 데이터 지점(public data point) 역할을 합니다.

**Live:** https://songyaeji.github.io/mcp-dashboard/

## Background

- 연구 본류(Remote MCP rug-pull detection)는 진행 중이라 비공개이며, 수집 데이터의 현황만 이 대시보드에서 공개합니다.
- 관련 선행 연구: "MCP 서버 메타데이터 기반 공급망 위협 탐지 연구 — Smithery Marketplace를 중심으로" (KCC 2026 발표).

## Structure

- `index.html` — 대시보드 본체 (정적 페이지, GitHub Pages 배포)
- `data.json` — 수집·집계된 MCP 생태계 메타데이터
